Shadow Cartels System-Centric Game UI/UX Design

As lead UI/UX designer for this complex Real-Time Strategy (RTS) game, my goal was to design a robust and scalable UI system that supports deep gameplay mechanics without overwhelming the user.

The Challenge: Scalable Complexity

Shadow Cartels is a data-driven strategy game. The main design challenge was displaying hundreds of real-time events (e.g., unit movements, economic data, interactive map information) simultaneously without overloading the user interface. Reducing interaction latency and cognitive load were key priorities.

Systemic Design Thinking

Scalability was solved by creating an atomic component-based UI Kit. Establishing the core principles of the design system (utilizing experience from industrial SCADA systems like WinCC OA) was crucial for consistency and rapid prototyping. This allowed for multiple iterations in parallel with evolving game mechanics.

AI-Augmented Workflow

We integrated "human in the loop" AI tools into the workflow to speed up prototyping of complex dashboards and maps, as well as generate visual options. AI-assisted testing helped validate various visual hierarchies, dramatically shortening the iteration cycle.

Key Module: Dynamic Map Data Visualization

The core of the game revolves around a custom, procedurally generated vector-based map displaying hundreds of moving elements. Instead of a static background image, we built interactive layers where the user must rapidly recognize critical information (e.g., resource shortages, enemy movements, territorial tensions).

  • Layer-based Scaling: Maps composed of separate SVGs were dynamically scaled and filtered based on the data selected by the user.
  • Visual Hierarchy: Using `data-visualization` principles, we distinguished between static background elements and dynamic, action-requiring data to minimize information overload.
  • Performance-First Prototypes: In-engine prototypes (built with the `data-icon` and Tailwind CSS technical foundation you use) helped validate technical feasibility in parallel with visual design development.

Results and Lessons Learned

This methodology and systems-based approach enabled the Shadow Cartels UI to be both functionally complex and user-friendly for its target audience. Interactive prototypes provided fast and effective feedback loops. Thanks to the scalable design system, integrating future content and features proved highly efficient.
