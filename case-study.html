The Brief

Create a minimalistic, non-intrusive UI for "Odyssey: Void", a first-person space survival game. The primary challenge was to convey critical survival stats (Oxygen, Suit Integrity, Fuel) without cluttering the breathtaking vistas of the procedural planets. The UI needed to feel "diegetic" — as if it were projected inside the player's helmet visor.

warning The Problem

  • Traditional health bars broke immersion.
  • Complex inventory management slowed down gameplay pace.
  • Text readability issues against bright stellar backgrounds.

check_circle The Goal

  • Context-aware UI that hides when idle.
  • Holographic visual language consistent with sci-fi lore.
  • Accessible color schemes for critical warnings.

UX & Player Flow

Before high-fidelity design, I mapped out the player's loop: Explore → Gather → Craft → Survive. The UI needed to support quick switching between these modes. We used FigJam to plot the user journey and identified friction points in the crafting menu.

HUD Conception

From wireframes to high-fidelity mockups. The concept evolved from a traditional "flat" UI to a curved, world-space UI that mimics a helmet projection.

Wireframe V1

Initial layout focused on corner-anchored stats, which felt too generic.

Final Design

Curved UI elements with chromatic aberration to simulate visor projection.
